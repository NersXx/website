# Persönliche Website

Beispiel-Projekt für eine persönliche Website mit vertikalen Personenen-Tiles.

## Setup

1. `git clone https://github.com/NersXx/website.git`
2. `cd website`
3. Dateien anpassen.
4. `git add . && git commit -m "Initial commit"`
5. `git push origin main` (oder master)

## Deployment

- In dein Webhosting-Panel https://...
- doc root für Hauptdomain auf `public_html` setzen
- Subdomain `alice.domain.de` -> `public_html/alice`, etc.
- SSL aktivieren (Let's Encrypt)
